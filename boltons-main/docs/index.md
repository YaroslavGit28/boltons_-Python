# Документация boltons-practical

Эта папка содержит расширенную документацию к учебной библиотеке `boltons_practical`.

## Навигация

- `usage.md` — установка, запуск тестов, быстрый старт
- `api.md` — публичный API пакета и реэкспорты из `boltons_practical.__init__`
- `fileutils.md` — утилиты файловой системы
- `dictutils.md` — утилиты для словарей
- `iterutils.md` — утилиты для итераторов/потоков данных
- `functional.md` — утилиты функционального стиля
- `wiki/Home.md` — учебная вики (контекст → механизмы → практика)
- `sources.md` — использованные источники
